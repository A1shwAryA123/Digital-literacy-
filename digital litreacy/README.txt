## Project Overview

This project is created as part of the Digital Literacy course. The objective of this project is to understand and apply essential digital skills required in today’s technology-driven world. As a Student Digital Ambassador, this project focuses on spreading awareness about digital literacy, safe internet practices, and building a professional online presence.

The project is divided into five tasks, each covering a different aspect of digital literacy. All tasks are organized in this repository along with supporting files, screenshots, and a detailed report.

---

## Task Summary

### Task 1 – Digital Literacy Infographic

Created a one-page infographic using Canva to explain the concept of digital literacy, safe internet practices, and online presence. The infographic highlights key points in a simple and visually appealing manner.

---

### Task 2 – Student Digital Portfolio

Set up and updated professional profiles on platforms such as GitHub and LinkedIn. Created a GitHub profile README and completed basic LinkedIn details to establish a digital presence.

---

### Task 3 – Platforms Exploration

Completed a beginner-level coding challenge on HackerRank and created a Google Form quiz on digital literacy awareness.

### Task 4 – Email Etiquette & Social Media Guide

Drafted two professional emails (assignment extension and internship request) and created a checklist of Do’s and Don’ts for responsible social media usage.

---

### Task 5 – Cybercrime Awareness

Prepared a case study on UPI fraud and created a prevention checklist to spread awareness about online safety and cybercrime prevention.
